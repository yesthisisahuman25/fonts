Exploring Different Font Types

Serif fonts have small lines at the ends of some characters to be stylish

  • Times New Roman
  • Georgia

Sans-Serif fonts do not. "Sans" means without.

  • Arial
  • Verdana

Monospace fonts ensure that all characters have the same space (width).

  • Courier New
  • Lucida Console

How do we read this?

font-family: "Arial", Arial, sans-serif;

The first font is the browser's first attempt. If it does not load, then the browser tries the second font. The third listed font is a backup in case none of the others load.

Suppose we want MORE CHOICES

Google has an API to allow us to link even more fonts! Just add an additional link tag with the font-family name after the question mark symbol.

Ex:
link href='https://fonts.googleapis.com/css?family=Sofia' rel='stylesheet'
In angle brackets in the head section, this would give you the font-family of Sofia.

Ex:
link href='https://fonts.googleapis.com/css?family=Acme' rel='stylesheet'
In angle brackets in the head section, this would give you the font-family of Acme.
